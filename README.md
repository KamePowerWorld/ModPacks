# かめぱわぁ～るどランチャー用Modパック管理リポジトリ

かめぱわぁ～るどランチャーに表示・配布されるMinecraft Modパックを管理するためのリポジトリです。

ここに Modパックを追加・更新すると、かめぱランチャー側に自動的に反映されます。

また、Google Colab 上で操作できる専用ノートブックを用意しており、
ローカル環境を用意しなくても、ブラウザだけで Modパックの追加作業が行えます。

## ⚠️ 注意点

かめぱランチャーは、  
「かめぱわぁ～るどで遊ぶ企画」向けに用意された専用ランチャーです。

そのため、以下の点にご注意ください。
- かめぱわぁ～るどで使用しない Modパックの追加は受け付けていません
- Modパック追加前に、内容や目的について事前相談をお願いします

企画やイベントで利用する場合は、相談の上で下記手順に従って準備を進めてください。

## 🚀 クイックスタート

### Google Colab で Modパックを追加する

以下のボタンから、Modパック追加用の Google Colab ノートブックを開けます。

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/KamePowerWorld/ModPacks/blob/master/Mod%E3%83%91%E3%83%83%E3%82%AF%E8%BF%BD%E5%8A%A0%E3%83%84%E3%83%BC%E3%83%AB.ipynb)

1. ボタンをクリックして Colab を開く
2. ノートブック内の説明に沿って入力・実行
3. Modパックがリポジトリに登録され、ランチャーに反映されます

※ Git や環境構築の知識がなくてもポチポチ押していくだけで操作できます。
